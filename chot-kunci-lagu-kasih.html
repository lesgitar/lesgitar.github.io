4. Fill in the blank with the right Forms of the verbs inbrackers to complete the following Englsh sentences inPresent Perfect tenseIwe (buy) -a new lamp.2. We (not (plan)- our holiday yet3 where (be / you)4. He (write)Five letters5. She Chot Isee)- him For a long time​


Isilah dengan bentuk kata kerja yang tepat ke dalam bentuk present perfect tense!

1). I have bought a new lamp.

2). We have not planned our holiday yet.

3). Where have you been?

4). He has written five letters.

5). She has not seen him for a long time.

Penjelasan :

Karena perintahnya menggunakan present perfect, maka rumus yang digunakan adalah.

(+) S + Have/has + V3 + O

(-) S + Have/has + not + V3 + O

(?) Have/has + S + V3 + O

Catatan :

Auxiliary verb HAVE digunakan untuk subject I/you/they/we/plural subject.Auxiliary verb HAS digunakan untuk subject orang ketiga tunggal seperti she/he/it/etc.

Perubahan kata kerja verb-1 menjadi verb-3 :

Buy => Bought => Bought

Plan => Planned => Planned

Be => was/were => Been

Write => Wrote => Written

See => Saw => Seen
