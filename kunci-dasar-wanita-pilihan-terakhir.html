29. Empat orang dipilih dari suatu kelompok yang terdiri dari 8 pria dan 6 wanita. Berapa peluang bahwa yang terpilih paling sedikit 2 wanita?


Kategori Soal : Matematika - Peluang
Kelas : XI (2 SMA)
Pembahasan :
Suatu kombinasi r elemen yang diambil dari n elemen yang tersedia adalah suatu pilihan dari r elemen tanpa memperhatikan urutannya (r ≤ n).
nCr = n!/(r! x (n - r)!)

Ruang sampel adalah himpunan semua hasil yang mungkin pada suatu percobaan yang dilambangkan dengan S.
Titik sampel adalah anggota-anggota dari ruang sampel.
Kejadian merupakan himpunan bagian dari ruang sampel atau bagian dari hasil percobaan yang diinginkan.
Jika S adalah ruang sampel dari suatu kejadian dengan setiap anggota S memiliki kesempatan muncul yang sama dan A suatu kejadian dengan A ⊂ S, maka peluang kejadian A adalah
P(A) = n(A)/n(S)
dimana n(A) merupakan banyak anggota dalam himpunan kejadian A dan n(S) merupakan banyak anggota dalam himpunan ruang sampel S.

Mari kita lihat soal tersebut.
Diketahui ada 14 orang dalam suatu kelompok yang terdiri dari 8 orang pria dan 6 orang wanita. Dipilih 4 orang dari kelompok tersebut dengan syarat paling sedikit ada 2 orang wanita.
Kita tentukan ruang sampelnya dahulu yaitu banyaknya kemungkinan dari suatu kelompok berjumlah 14 orang dipilih 4 orang adalah
n(S)
= ₁₄C₄
= 14!/(4! x (14 - 4)!)
= 14!/(4! x 10!)
= (14 x 13 x 12 x 11 x 10!)/(4 x 3 x 2 x 1 x 10!)
= 7 x 13 x 11
= 1001
Kemudian, banyaknya suatu kejadian dari 8 orang pria dan 6 orang wanita dipilih 4 orang dengan syarat paling sedikit 2 orang wanita adalah
n(A)
= ₈C₂ x ₆C₂ + ₈C₁ x ₆C₃ + ₈C₀ x ₆C₄
= 8!/(2! x (8 - 2)!) x 6!/(2! x (6 - 2)!) + 8!/(1! x (8 - 1)!) x 6!/(3! x (6 - 3)!) + 8!/(0! x (8 - 0)! x 6!/(4! x (6 - 4)!)
= 8!/(2! x 6!) x 6!/(2! x 4!) + 8!/(1! x 7!) x 6!/(3! x 3!) + 8!/(0! x 8!) x 6!/(4! x 2!)
= (8 x 7 x 6!)/(2 x 1 x 6!) x (6 x 5 x 4!)/(2 x 1 x 4!) + (8 x 7!)/(1 x 7!) x (6 x 5 x 4 x 3!)/(3 x 2 x 1 x 3!) + 1 x (6 x 5 x 4!)/(4! x 2 x 1)
= 4 x 7 x 3 x 5 + 8 x 5 x 4 + 3 x 5
= 420 + 160 + 15
= 595
Jadi, banyaknya peluang dari suatu kelompok berjumlah 14 orang yang terdiri dari 8 orang pria dan 6 orang wanita dipilih 4 orang dengan syarat paling sedikit 2 orang wanita adalah 
P(A) = n(A)/n(S)
⇔ P(A) = 595/1001

32. Algoritma yang mencari informasi berdasarkan kata kunci mulai dari awal array hingga akhir array adalah


Jawaban:

Salah satu algoritma yang dapat digunakan untuk mencari informasi berdasarkan kata kunci dari awal array hingga akhir array adalah algoritma Linear Search. Berikut adalah pseudocode untuk algoritma Linear Search:

```

1. Inisialisasi posisiAwal = -1 (berarti kata kunci tidak ditemukan)

2. Untuk setiap elemen arr[i] dalam array:

a. Jika arr[i] sama dengan kata kunci:

i. Set posisiAwal = i

ii. Hentikan pencarian

3. Kembalikan posisiAwal

```

Dalam algoritma ini, pencarian dimulai dari awal array dan berlanjut hingga akhir array. Jika kata kunci ditemukan, posisi awal diperbarui dengan indeks di mana kata kunci ditemukan. Jika kata kunci tidak ditemukan, posisiAwal tetap -1.

Berikut adalah contoh implementasi algoritma Linear Search dalam bahasa Python:

```python

def linear_search(arr, kata_kunci):

posisi_awal = -1

for i in range(len(arr)):

if arr[i] == kata_kunci:

posisi_awal = i

break

return posisi_awal

```

Anda dapat memanggil fungsi `linear_search` dengan memberikan array dan kata kunci yang ingin dicari sebagai argumen. Fungsi akan mengembalikan indeks pertama di mana kata kunci ditemukan, atau -1 jika kata kunci tidak ada dalam array.

Contoh penggunaan:

```python

array = [4, 2, 7, 1, 9, 5]

kata_kunci = 7

posisi = linear_search(array, kata_kunci)

print("Kata kunci ditemukan pada indeks:", posisi)

```

Output:

```

Kata kunci ditemukan pada indeks: 2

```

Dalam contoh di atas, kata kunci "7" ditemukan pada indeks ke-2 dalam array.
