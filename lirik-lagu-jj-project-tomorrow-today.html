32. Today is Monday tomorrow is​


Telling the Days

Today is Monday, tomorrow is​ ...

=> Tuesday

PEMBAHASAN:

Hari ini hari (Monday) Senin, besok adalah hari (Tuesday) selasa.

Day berarti hari dalam bahasa inggris, yaitu setiap periode selama 24 jam, dimulai dari tengah malam, yang terbagi dalam satu minggu, bulan, atau tahun sesuai dengan rotasi bumi terhadap porosnya.

Perhatikan nama-nama hari dalam bahasa inggris dan cara membacanya berikut ini!

Minggu => Sunday => /ˈsʌndeɪ/  Senin => Monday => /ˈmʌndeɪ/  Selasa => Tuesday => /ˈtjuːzdeɪ/  Rabu => Wednesday => /ˈwɛnzdeɪ/  Kamis => Thursday => /ˈθəːzdeɪ/  Jumat => Friday => /ˈfrʌɪdeɪ/  Sabtu => Saturday => /ˈsatədeɪ/  

Perhatikan pertanyaan yang biasa digunakan untuk menanyakan hari berikut ini!

1. A : What day is it today? (Hari apa sekarang?)  

   B : It is Friday. (Ini hari Jum'at.)  

2. A : What day is tomorrow? (Besok hari apa?)  

   B : Tomorrow is Saturday. (Besok hari Sabtu.)  

3. A : What day is the day after tomorrow? (Hari apa sesudah besok?)  

   B : It is Sunday. (Itu hari Minggu.)  

4. A : What day was it yesterday? (Kemarin hari apa?)  

   B : It was Thursday. (Kemarin hari Kamis.)  

5. A : What day was the day before yesterday? (Hari apa sebelum kemarin?)      B : It was Wednesday. (Itu hari Rabu.)
