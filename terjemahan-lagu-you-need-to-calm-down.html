14. subjek dari kalimat To avoid them, you will need to double down on the same pandemic precautions that have kept you safe so far


Subjeknya you

Them adalah kata objek.

Disitu diminta subjek you melakukan sebuah tindakan. So jawabanya you

Jadikan jwbn terbaik,yh kalau bener sih hehe

Jawaban:

To avoid them, you will need to double down on the same pandemic precautions that have kept you safe so far.

Subject: You

Verb: Need

Object: The same pandemic precautions

Penjelasan:

"To avoid them" disini hanyalah merupakan "keterangan // tambahan untuk kalimat"

[sebenarnya, "to avoid them" disini hanyalah subordinate clause, main clause terletak di "you will need to double down on the same pandemic precautions that have kept you safe so far" - untuk pemula bisa skip part ini, ini bisa dipelajari nanti]

subject-nya adalah kamu.

verb-nya: adalah need (harus)

object-nya itu tindakan pencegahan pandemi yang sama (the same pandemic precautions)

Lain-lain:

semoga membantu ;)

semoga kakak bisa mengerti, maaf kalau penjelasannya kepanjangan.

mapel: B. Inggris

kelas: 1 SD // 2 SD (kalau kurikulum beda, bisa chat saya di kolom komen, bisa saya ganti)

topik: Structure of a sentence
